Ejyle.DevAccelerate.Core Namespace

The Ejyle.DevAccelerate.Core namespace contains types that provide core functionality and foundation for applications that are built on top of the DevAccelerate framework.
Classes
  ClassDescription
DaApplicationContext
Represents the APIs to bootstrap a DevAccelerate application. This class cannot be inherited.
DaApplicationContextExtensions
An empty class to which extensions can be added by anyone and then access them through the Extensions property of the DaApplicationContext static class.
EntityBase
Represents the base class for an entity.
EntityBaseTKey, TOptionalKey
Represents the base class for an entity.
EntityManagerBaseTKey, TOptionalKey, TEntity, TRepository
Represents the APIs for entity management and uses the entity repository internally.
Interfaces
  InterfaceDescription
IDatebaseContext

IEntityTKey, TOptionalKey
Defines the properties for an entity.
IEntityRepositoryTKey, TOptionalKey, TEntity
Defines the methods for entity management.
