Ejyle.DevAccelerate.Core.Configuration Namespace

The Ejyle.DevAccelerate.Core.Configuration namespace contains types that implement basic configuration management.
Classes
  ClassDescription
DaConfigurationSection
Provides the base methods for a configuration section implementation.
DefaultConfigurationSource
Represents the default implementation of configuration source which is either web.config or app.config of the application.
ExtendedPropertyConfigurationElement
Represents additional properties for a named configuration element.
ExtendedPropertyConfigurationElementCollection
Contains a collections of the ExtendedPropertyConfigurationElement objects.
FileConfigurationSource
Represents a file-based configuration source.
NamedConfigurationElement
Represents a named property for a provider configuration.
NamedConfigurationElementCollectionTNamedConfigurationElement
Contains a collections of the NamedConfigurationElement objects.
ProviderConfigurationSection
Defines the properties for a provider configuration.
Interfaces
  InterfaceDescription
IConfigurationSource
Defines the methods for a configuration source.
