Overview

DevAccelerate Core is the core library of the DevAccelerate Framework. Every other DevAccelerate library depends on DevAccelerate Core. DevAccelerate core provides some of the core functionalities such configuration management, caching, exception handling, and log management.

DevAccelerate does not impose any particular implementation of caching, logging, or exception handling. It uses a well-established provider model that can be used to plug other implementations and libraries such as Microsoft Enterprise Library. The obvious benefit of this flexibility is that you can switch to any other underlying framework such as Log4Net by simply making a configuration change without making any modifications to your code whatsoever.
